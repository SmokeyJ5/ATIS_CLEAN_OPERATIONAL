# ATIS CLEAN OPERATIONAL v4.0 — Professional Workstation Architecture Foundation

## Added
- Event bus foundation
- Symbol selected events
- Watchlist changed events
- Professional Watchlists tab
- JSON persisted watchlists
- Default watchlists:
  - Day Trading
  - Silver Miners
  - Gold Miners
  - Copper Uranium
  - Favorites
- Workstation Architecture tab
- Event Bus Activity panel

## Architecture Direction
ATIS now has the foundation to move from one large directly wired app into an event-driven workstation.

## Preserved
- One global search bar
- All-symbol live lookup
- Live Market Workstation reports
- Paper trading remains simulated only
- No real broker execution
